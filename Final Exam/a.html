What is the difference of HTML? and CSS?

HTML is a markup language that we use to create elements onto a web site page. You can create content and an entire page alone with just HTML. CSS is used to style that HTML, it is a style sheet language used to specify the formatting of a page.

What is the relationship between HTML and CSS

HTML structures a page and CSS modifies the presentation. Therefore they work together to create a beautiful, fluid, and well decorated page that couldn't be done entirely on their own. They need each other like a child needs a mother and a father to be conceived.

Three ways to add CSS to a web page?

  1. With an external style sheet, or an externale page or file.
  2. Embedding it in the same page at the top of the page within the head tags and using the style tags.
  3. CSS can be added in line, adding it into the same line of the actual tag that you want to style that is within the page itself.

What is the difference between id and class in CSS?

Class is used to create a class or group of elements that can all use the same type of CSS styling either embeded or in an external style sheet. Id is used to give a name to a single HTML element that can be styled elsewhere also, but it's not used to signify a class or group of elements togheter. Id is called using the '#' or hash tag in CSS and class is used by a '.' or period.
